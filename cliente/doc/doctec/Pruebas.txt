Pruebas
=======

* **Pruebas iniciales con Mocks**: Para facilitar el testeo de la aplicación independientemente del servidor, se creo un mock del servidor en python que simula su funcionalidad. A su vez, este fue previamente testeado realizando http requests mediante el plugin de chrome `postman <https://chrome.google.com/webstore/detail/postman-rest-client/fdmmgilgnpjigdojojpjoooidkmcomcm>`_ .

* Posteriormente, con el proyecto más avanzado, se probó toda la aplicación utilizando el servidor real.

Pruebas de integración con el servidor
======================================

Para las siguientes pruebas con el servidor, se utilizaron básicamente dos dispositivos.

* Un celular Moto E de segunda generación con Android Lollipop, registrando un usuario de nombre "poly" y nombre para mostrar "polaris"
* Una tablet Kelyx con Android Jelly Bean, registrando un usuario de nombre de usuario y nombre para mostrar iguales, "Venecia"
